## Introduction to Linux & Bash

# Shell 

A shell is a program that interprets commands from the user and communicates with the operating system kernel.

 Think of it like this:
 You (type command) → Shell (interprets) → OS Kernel (executes) → Output (returns to you)

 # Bash

 Bash(Bourne Again Shell) is the most popular shell on linux syatems.


check bash and the version  

```bash
 echo $SHELL
 bash --version
```
# Terminal 

It is the window from where we gives commands 

# Common terminal shortcuts:
Ctrl + C     # Cancel current command
Ctrl + Z     # Suspend current process
Ctrl + D     # Exit terminal / End of file
Ctrl + L     # Clear screen (or type 'clear')
Ctrl + A     # Move to beginning of line
Ctrl + E     # Move to end of line
