# React Commands

React is a JavaScript library for building component-based user interfaces. Most modern React projects use Vite, Next.js, or another framework for bundling, routing, and development tools.

## Create Apps

```bash
npm create vite@latest my-app -- --template react
pnpm create vite my-app --template react
npx create-react-app my-app
pnpm dlx create-react-app my-app
```

## Common Scripts

```bash
npm run dev
npm run build
npm run preview
npm test
pnpm dev
pnpm build
pnpm preview
pnpm test
```

## Script Definitions

- `dev` - Start the local development server.
- `build` - Create optimized production files.
- `preview` - Preview a production build locally, common in Vite apps.
- `test` - Run the project test command from `package.json`.

## Install Common Packages

```bash
npm install react-router-dom
pnpm add react-router-dom
npm install axios
pnpm add axios
npm install @tanstack/react-query
pnpm add @tanstack/react-query
npm install zustand
pnpm add zustand
npm install react-hook-form zod
pnpm add react-hook-form zod
npm install tailwindcss @tailwindcss/vite
pnpm add tailwindcss @tailwindcss/vite
```

## Useful React Packages

- `react-router-dom` - Client-side routing for single page apps.
- `axios` - HTTP client for API requests.
- `@tanstack/react-query` - Server-state fetching, caching, and synchronization.
- `zustand` - Small state management library.
- `react-hook-form` - Form state and validation helper.
- `zod` - Schema validation for form data and API payloads.
